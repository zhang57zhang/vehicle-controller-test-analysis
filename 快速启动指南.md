# 快速启动指南

## 🚀 三步快速启动

### 第一步：检查环境

打开命令提示符（CMD），输入：

```bash
python --version
node --version
```

如果显示版本号，环境OK。如果提示"不是内部或外部命令"，请先安装：
- Python：https://www.python.org/downloads/
- Node.js：https://nodejs.org/

### 第二步：一键启动

**推荐方式（无乱码）：** 双击运行 `一键启动_新版.bat` 文件

**备用方式：** 双击运行 `一键启动.bat` 文件

### 第三步：访问系统

浏览器访问：http://localhost:5173

## ❓ 遇到问题？

### 问题1：启动脚本出现乱码
**解决方案：** 使用 `一键启动_新版.bat` 替代 `一键启动.bat`

### 问题2：端口被占用
运行 `停止服务.bat`，然后重新启动

### 问题3：无法访问
检查防火墙是否阻止了Python和Node.js

### 问题4：服务窗口自动关闭
手动进入对应目录启动服务，查看错误信息

### 问题5：PowerShell执行策略错误
**解决方案：**
1. 右键点击 `一键启动_新版.bat`
2. 选择"以管理员身份运行"
