# 快速启动指南

## 🚀 三步快速启动

### 第一步：检查环境

打开命令提示符（CMD），输入：

```bash
python --version
node --version
```

如果显示版本号，环境OK。如果提示"不是内部或外部命令"，请先安装：
- Python：https://www.python.org/downloads/
- Node.js：https://nodejs.org/

### 第二步：一键启动

双击运行 `一键启动.bat` 文件

### 第三步：访问系统

浏览器访问：http://localhost:5173

---

## 📋 服务地址

| 服务 | 地址 |
|------|------|
| 前端界面 | http://localhost:5173 |
| 后端API | http://localhost:8000 |
| API文档 | http://localhost:8000/api/docs |

---

## 🛑 停止服务

双击运行 `停止服务.bat` 文件

---

## ❓ 遇到问题？

### 问题1：端口被占用
运行 `停止服务.bat`，然后重新启动

### 问题2：无法访问
检查防火墙是否阻止了Python和Node.js

### 问题3：服务窗口自动关闭
手动进入对应目录启动服务，查看错误信息
